# Calificador de CV (Next.js)

Aplicación de RRHH para evaluar CVs de forma rápida con una puntuación sobre 100 y feedback accionable.

## Levantar proyecto

```bash
npm install
npm run dev
```

Abrir `http://localhost:3000`.

## Qué hace

- Evalúa el texto del CV contra habilidades base.
- Estima nivel competitivo para el rol.
- Señala fortalezas, brechas y recomendaciones.
